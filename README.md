# Tax Marketplace

Claude Code plugin marketplace for AI-powered tax filing tools.

## Plugins

| Plugin | Description |
|--------|-------------|
| [super-cpa-us-tax](https://github.com/Fangyi-Chen/super-cpa-us-tax) | AI tax filing assistant — interview, W2/1099 import, federal tax calculation, Form 1040 PDF generation |

## Installation

```
/plugin marketplace add Fangyi-Chen/tax-marketplace
/plugin install super-cpa-us-tax@tax-marketplace
```
