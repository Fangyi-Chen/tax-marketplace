# Tax Marketplace

Claude Code plugin marketplace for AI-powered, open-source tax filing tools.

## Plugins

| Plugin | Description |
|--------|-------------|
| [super-tax-agent](https://github.com/Fangyi-Chen/SuperTaxAgent) | AI tax filing assistant — interactive interview, W2/1099 import, federal tax calculation, Form 1040 PDF generation. Self-prep use only. |

## Installation

```
/plugin marketplace add Fangyi-Chen/tax-marketplace
/plugin install super-tax-agent@tax-marketplace
```

## Disclaimer

Plugins in this marketplace are provided **for informational purposes only** and do not constitute tax, legal, or accounting advice. They are intended for individuals preparing their own returns. Consult a licensed tax professional for advice specific to your situation.
